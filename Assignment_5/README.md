# Quicksort Algorithm: Deterministic and Randomized

This repository contains Python implementations and performance experiments for:

- Deterministic Quicksort
- Randomized Quicksort
- Empirical timing comparison (matplotlib graph)





### Quicksort
- Pivot = first element
- May degrade to O(nÂ²) on sorted or reverse-sorted input

### Randomized Quicksort
- Pivot chosen uniformly at random
- Avoids worst-case behavior with high probability
- Expected running time: O(n log n)

### Empirical Comparison
- Tests both algorithms on various input sizes
- Generates a performance plot using matplotlib
