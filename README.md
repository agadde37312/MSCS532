# Assignment 3 - Algorithm Efficiency & Scalability

## Contents
- randomized_quicksort.py : randomized and deterministic quicksort implementations
- hash_table_chaining.py : hash table with chaining and dynamic resizing
- benchmarks.py : scripts to run empirical comparisons and output CSV / text results


## How to run
1. Ensure Python 3.8+ installed.
2. Run benchmarks:
   python benchmarks.py
3. Inspect quicksort_results.csv for quicksort timings and hashtable_results.txt for hash table timings.

## Notes
- Benchmarks use moderate sizes; increase sizes in `benchmarks.py` if you have sufficient memory and CPU.
- For reproducibility, set `random.seed(...)` in `benchmarks.py`.
